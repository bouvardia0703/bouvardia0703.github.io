BZOJ4538 线段树,堆,树链剖分

Feb 26, 2019

巧妙的转化.

Description

有若干次路径标记/撤销操作.

每次询问一个位置, 没有经过这个这个位置的路径标记最大值.

Solution

每个节点维护一个堆, 为完全没有经过这个区间的路径标记最大值.

堆要惰性删除, 不过有奇技淫巧(通过特殊手段避免MLE).

每次更新, 去区间的补集在线段树上做就可以了.
