Feb,2019 TOEFL考试记

Feb 24, 2019

又双叒叕考托福去了.

最终还是没定成出国的事, 所以这次考试又变成体验场了, 反正到大学的时候也早就失效了.

头一天突然意识到几天看起来学了什么东西其实什么都没学, 突然慌张, 赶紧拿出新东方熟悉了一波口语答题范例和缓存的写作课, 事实证明除了熟悉了已经阔别半年的题型之外并没有什么卵用. 打水漂打了两万块. 有用的课还全都没听, 也来不及了, 唉呀呀.

一直快进到12点多终于是睡了, 幸好睡了机房, 我居然不知道学校是周末作息, 要是真在考试日七点起床我绝对慌得要死. 然后磨磨蹭蹭起床溜出了学校, 还被母上嫌晚了. 早饭也没吃, 走之前跑去机房喝了一口水.

垃圾滴滴, 你站出来啊! 你怎么怕了! 总是忘记换滴滴的手机号, 这次终于换过来了, 然后就GG了. 实名认证未成年不让打车. 绝望. 好在学校这种地方总会有出租车守着的, 然后就出发了.

周末没有堵车美滋滋, 车上本来想泛听半套TPO结果Conversation死活下载不来, 听了一篇Lecture就摸了, 一边听歌一边看了要来的高分替换词最后也没有用上几个.

终于到了. 等等, 为什么全是什么艺术生培训还有一堆小哥哥小姐姐, 我怕不是走错了(怀疑人生), 啥这学校在办艺考??? 不会真走错了吧(怀疑人生x2). 吓得赶紧摸出地址看了看应该没错, 又凑过去看了发平面图, 找到楼了. 艺考生都去另一个方向了, 于是一个人走在小路上连个牌都没有, 十分紧张. 体验极差x1. 上次去那个地方好歹从进门就有牌吧.

进了楼还有别人一起, 看了眼楼层牌写着托福考场, 总算稳了. 还没人签到, 摸出手机和词汇小册子想抢救一发, 结果看公众号的预测, 就YY了一道独立写作题就把时间用光了(sad. 排队register, 一堆小哥手里拿着确认信, 我没拿, 反正我考过强行不慌. 把大衣书包之类的都锁起来了, 突然发现穿的是校服, 又给学校丢脸了. 抄一发保密协议准备入场.

体验极差x2. 临时身份证爆炸了, 他们怎么也核验不过我的照片, 就被等到了最后. 还把电脑搞死机了, 行吧我头大. 最后还是进去了. 这地方挡板真矮, 不过倒是和我没啥关系. 开考吧.

故意拖了一会, 还是觉得太没意思了, 我准备做阅读了还有人在Describe他的city. 一开Direction, 1h20min, 哦豁加试. 不过这一般意味着听力不会加试了, 感觉海星. 上次阅读有点紧, 这波要快做, SAT那个方法用起来.

  • 英国经济发展因素
  • 鸟类的头部的一些特性
  • 欧洲城市设计的演变
  • 太阳系行星的产生过程

顺序记不得了, 这次换了一种读法, 每篇文章先只看段首句, 大概搞到一个脉络, 然后直接开始做题. 除了有些题比较犹豫浪费时间外, 感觉确实快了, 虽然还是比不过大神, 不过20min的时间基本15min就能做完了.

然后是听力, 果然没有加试, 而且不想去卫生间, 感觉不错.

  • C 想从剧院系转到音乐系
  • C 英语120太难了, 想放弃, 教授告诉他他错了
  • L Room Temperature Ion Liquid的特性和用途(Che)
  • L Landscape画家TC的一些事迹(Art History)
  • L 一卷地图, 和美洲为什么叫美洲(Geo)
  • L 忘记了

最后一篇听力还是有点累了. 休息休息.

体验极差x3. 这考场休息时间结束了没有人来带你, 要自己提前回去等着! 我都开始做口语了还有人在听力, 等了一会, 觉得不该等, 趁大家没开始说话赶紧开了.

  • S1 快餐店和街边摊有什么优点或缺点?
  • S2 孩子应不应该养宠物长大?
  • S3 一个大四学生觉得不应该让大一大二的去工作workshop, 因为他们用不到, 而且可以上网. 听力觉得不对, 因为他们有这个需要, 而且网上信息太少, 去workshop可以得到建议(这我没意识到).
  • S4 讲人对未来预测的不准确性, 特别是当结果已经发生的时候, 人们会有”他当时就是这么想的”的感觉. 叫Hindsight Bias. 听力举了一个出书的朋友的例子.
  • S5 要去Jordan留学, 需要练conversation, 但是时间冲突, 可以上网课或者和朋友练. 选和朋友连, 节省时间, 增进友谊.
  • S6 沙漠动物保护自己的两种方式, 快速走, 多休息, 还有贴地运动, 少影子. 都是lizard的例子.

口语又跪了点, 有个题最终还是干等了一会. 还有题没录完.

综合写作是不应该吃gluten, 因为人的消化系统不适应, 会胃痛胃疲倦, 还有高血糖引发brain inflammation进而引发memory problem, 然后听力驳斥了.

独立写作, 该不该让所有大一新生上学习方法课, 应该, 对教学有帮助, 可以更好地完成教学任务, 对学生将来有帮助. 来不及让步了.

综合稍微多点, 独立没够400感觉要凉啊.

我居然是一个区第三个出来的, 出来和母上商量了一下集训的事情, 买了个饼吃就回学校了. 要是能继续该多好.

感觉口语和写作还是没什么大的提升, 唉. 上100都难了.

家那边也开学了, 走一步是一步吧.
