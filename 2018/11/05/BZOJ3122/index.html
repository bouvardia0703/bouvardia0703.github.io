BZOJ3122 SDOI2013 随机数生成器

Nov 05, 2018

BSGS + Exgcd

Table of Contents

  1. Description
  2. Solution

Description

给定一个递推方程：$X_{i+1} = (aX_i + b)\,mod\,p$, 求最小的$i$, 使得$X_i = t$.

Solution

讨论$a = 0, a = 1, a \ge 2$.

分别是特判, exgcd, BSGS + exgcd.
