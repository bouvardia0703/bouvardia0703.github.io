BZOJ4592 SHOI2015 脑洞治疗仪

Nov 05, 2018

线段树。

Table of Contents

  1. Description
  2. Solution

Description

维护一个01序列, 支持查询区间1的个数, 区间置0, 取出区间里所有的1并填入某个区间, 优先填靠前的0, 多了就浪费.

Solution

首先看见区间操作肯定要线段树的. 区间查询和置0都好做, 主要是操作3.

操作3的第一步是取出1, 这个和区间查询同理.

如果发现要浪费, 就直接区间置1.

关键就在于每次找一段最长0了, 这个可以二分啦. 有时区间头是1, 那么我们还需要跳过一段最长1. 这两个二分都可以做.
